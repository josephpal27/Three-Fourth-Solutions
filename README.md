# Three Fourth Solutions — Company Website

Official company website for Three Fourth Solutions, designed and developed as a modern, responsive, and performance-focused web platform.

This project was built and maintained independently by me as the sole Frontend Developer, handling UI design implementation, responsiveness, and user experience optimization across devices.

---

## Project Overview

The website represents the digital presence of Three Fourth Solutions and includes key business pages such as:

* Home
* Verticals
* Press & Media
* Careers
* Contact
* PR
* Studios
* Finance
* Admin login interface
* Custom error pages
* Interactive elements and responsive layouts

The goal was to create a clean, professional, and accessible web experience aligned with business branding.

---

## Features

* Clean and structured UI layout
* Performance-optimized static pages
* SEO-friendly structure
* Contact and career form integration with backend
* Admin access interface
* Social sharing support
* Interactive animations

---

## Tech Stack

* HTML5
* CSS3
